Welcome to American Sign Language API


The American Sign Language (ASL) is a popular Sign Language used for Sign Language communication. The ASL dataset is a popular dataset for ASL. This dataset is popularly used for Image Processing/Computer Vision tasks that involve Sign Language detection.
This API allows for the use of a CNN model based on MobileNet architecture built using the ASL dataset.
The following are the API Endpoints and how they can be used.

End Point Method Description Parameters Response
/login POST Allows Administrator to log in to their dashboard Username : String
Password : String		 True or False - Boolean
/student/login POST Allows Student to log in to their dashboard Username : String
Password : String		 True or False - Boolean
/student GET Allows to view list of all students given the administrator has logged in. A token will be used to verify the logged in user. JSON Object - Array of all Students and their details
Empty array if there are no students.
/student POST Allows to create a new Student in the database by passing the details of the Student. username, password, name, email, year True or False - Boolean
Whether the Student has been successfully created or not.
/student?id=(id) GET Allows to view an individual student given their ID. id : integer JSON Object - Student Details
/student?id=(id) PUT Allows to update the details of an existing student given the details with which the student needs to be updated. id : integer, username : string, password : string, name : string, email : string, year : string True or False - Boolean
Whether the Student was updated successfully or not.
/student?id=(id) DELETE Allows to delete an existing student given the ID of the student. Token : String, id : integer True or False - Boolean
Whether the student was deleted successfully or not.
/student/exam?studentid=(studentid)&attempt=(attempt) POST Allows to insert a new exam result taken by a student. Need Attempt Number and StudentID. studentid : integer, attempt : integer, correct : integer, incorrect: integer True or False - Boolean
Whether the Exam for the Student was inserted successfully.
/student/exam?studentid=(studentid)&attempt=(attempt) GET Get Details of all the attempts of the student for that exam - Student ID and Exam ID. studentid : integer, attempt : integer JSON Object - Array of all the exams taken by the student with all the details.
/student/exam?id=(id)&studentid=(studentid)&attempt=(attempt) GET Get Details of the Specific Attempt of the Specific Exam (attempt) by that Specific Student (StudentID). id : integer, studentid : integer, attempt : integer JSON Obhect - Details of the specific attempt of that exam taken by that student.
/student/exam?id=(id)&studentid=(studentid)&attempt=(attempt) PUT Update the details of the exam with the given ID taken by the student with the given ID. id : integer, studentid : integer, attempt : integer True or False - Boolean
Whether the Update was successful
/student/exam?studentid=(studentid)&attempt=(attempt) DELETE Delete all attempts of the exam taken by the student. studentid : integer, attempt : integer True or False - Boolean
Whether Deletion of records was successful
/student/exam?id=(id) DELETE Delete the instance of the exam taken up by the student. id : integer True or False - Boolean
Whether the Deletion of the record was successful
/user POST Insert New User into the Database. username : string, password : string, name : string, email : string, role : integer True or False - Boolean
Whether the new user has been successfully added or not.
/user?id=(id) GET Get user with the given user id. id : integer JSON Object - The user object.
/user GET Get a list of all users using the platform. JSON Object - Array of all the users using the platform.
/user?userid=(userid) PUT Update an existing user with the given User ID. userid : integer, username : string, password : string, name : string, email : string, role : integer True or False - Boolean
Whether the User with the given ID was successfully updated or not.
/user?userid=(userid) DELETE Delete a User with the given User ID. userid : integer True or False - Boolean
Whether the User with given ID was successfully deleted or not.
/messenger POST Add a New message sent to another user by a student. studentid : integer, userid : integer, message : string True or False - Boolean
Whether the Message was successfully added or not.
/messenger?studentid=(studentid) GET Retrieve all the messages sent and received by the student with the given student id. studentid : integer JSON Object - Array of all messages belonging to that student.
/messenger?userid=(userid) GET Retrieve all messages sent and received by the user with the given user id. userid : integer JSON Object - Array of all messages belonging to that user.
/messenger?userid=(userid)&studentid=(studentid) GET Retrieve all messages exchanged between the user with the userid and student with the studentid. userid : integer, studentid : integer JSON Object - Array of all messages exchanged between that user and student.
/messenger?id=(id) PUT Update an existing message with the given id. id : integer, message : string True or False - Boolean
Whether the message was successfully updated or not.
/messenger?id=(id) DELETE Delete an exisitng message with the given id. id : integer True or False - Boolean
Whether the message was successfully deleted or not.
/messenger?studentid=(studentid) DELETE Delete all messages sent by the student with the given student id. studentid : integer True or False - Boolean
Whether the messages were successfully deleted or not.
/messenger?userid=(userid) DELETE Delete all messages sent by the user with the given user id. userid : integer True or False - Boolean
Whether the messages were successfully deleted or not.
/messenger?userid=(userid)&studentid=(studentid) DELETE Delete all messages exchanged between the user with the userid and student with the studentid. userid : integer, studentid : integer True or False - Boolean
Whether the messages were successfully deleted or not.
/predict POST Get Prediction on the image. file : file Return Predicted Class as Json Object.
